# RMSNorm
> RMSNorm = Root Mean Square Normalization(均方根归一化) 
## 为什么要归一化
在神经网络训练时，会出现：  
- 数值太大，导致梯度、激活爆炸
- 数值太小，导致梯度、信号消失

**目标**是`控制信息尺度`，使后续计算更稳定、更好训练：   
- 稳定前向传播
- 稳定反向传播
- 减少对`初始化`和`超参数`的敏感性
- 让深层网络更容易训练  

### 对`初始化`和`超参数`的理解:  
- 初始化（initialization）
  - 网络训练时，权重是**随机初始化**
  - 若网络初始化不理想，会导致训练崩溃、不收敛
  - 减少对初始化的敏感性 → 换不同的随机初始化，模型照样能训练好，结果相差不大
- 超参数（hyperparameters）
  - 学习率、batch size、正则化系数、网络深度等
  - 有的模型学习率一大，就不收敛
  - 减少对超参数的敏感性 → 超参数不用精调，在一个比较宽的范围里都能 work

## 将什么数据归一化
把`网络内部的中间激活`归一化，而不是输入数据  
- 某一层的hidden state
- 某个token对应的向量
- 残差连接之后的向量  
> 即，归一化**每个样本、每个 token 在当前层里的特征向量**  

### 对hidden state（隐藏状态）的理解：  
模型在某一层、某一时刻，对输入信息的“内部表示”。
- 它不是输入（input）  
- 也不是最终输出（output）
- 而是中间过程中的内部变量  
- 对用户来说是“看不见的”，所以叫 hidden    

例如一个 token 的表示是：
$$x \in \mathbb{R}^d$$
RMSNorm 就是在这个向量的`特征维`上做操作，而不是在 batch 维上做。

### 特征维是什么意思
例如，一个batch的数据：
``` python
x.shape = (batch_size, seq_len, d)
```
| 维度          | 含义                      |
| :----------:  | :-------------------:     |
| batch_size    | 一次喂多少句子             |
| seq_len    | 每句话多少 token             |
| d          | 每个 token 的向量长度（特征数）|
`特征维`就是最后一维 `d`  
``` python
x[i][j] = # 一个 token 的向量（长度 d）
```
RMSNorm做的是：
- 在`特征维`对每一个 token 的向量，单独做归一化；
- 不关心其他样本
- 不关心其他token
- 不跨 token、不跨 batch
  



## 归一化后的数据变成了什么样？
### 如果是 LayerNorm：
它会把向量变成“`均值为 0`、`方差为 1`”附近的状态。也就是：
- `平均值被拉到 0`
- 尺度被统一到 1
- 再乘上可学习参数$\gamma$，恢复一定表达能力
### 如果是 RMSNorm：
它会把向量的均方根（RMS）拉到 1 附近。也就是：
- `均方根被拉到 1`
- 但`均值不变`
- 再乘上可学习参数$\gamma$，恢复一定表达能力
- 向量的方向大体不变
- 每个分量之间的相对比例大体不变
- 只是整体大小被缩放到一个标准范围
 
> LayerNorm：把“位置”和“大小”都重新整理  
> RMSNorm：只把“大小”整理好，尽量不动“位置信息”

### 为什么“减均值”会改变方向？  
- 如向量：[2, 4]
- 均值：$\mu = \frac{2+4}{2} = 3$
- 减均值后：[2-3, 4-3] = [-1, 1]  
    |      | 原来  | 现在   |
    |:---: | :---: | :---: |
    | 比例 | 1 : 2 | -1 : 1 |
    | 方向 |   ↗  |   ↖   |
- `本质原因`：
  - LayerNorm 做平移：$x → x-\mu$
  - RMSNorm 做缩放：$x → c*x$
- `背后思想`:
  - LayerNorm 认为均值=噪声
  - RMSNorm 认为均值可能是信息的一部分，不该随便动  

RMSNorm 让每个 token 的表示都处在差不多的能量水平上，但**保留**原有语义方向。  
LayerNorm 则**改变**了语义方向。

## 归一化在算法链路中的什么位置？发挥什么作用？
归一化经常出现的位置：**复杂变换之前**  
### 在transformer的位置
在transformer里，常出现在`残差块`内部，常见Pre-Norm结构：  
$$x→Norm→Attention/FFN→+x$$  

### 作用
- 第一层：给子模块一个稳定输入
  - 注意力层、MLP 层都更喜欢“范围稳定”的输入。
  - 否则同一个模块今天看到很大数值，明天看到很小数值，训练会很飘。

- 第二层：保护残差链路
  - Transformer 很依赖残差累加。
  - 如果每层输出尺度不受控，残差会越加越大，或者越加越乱。
  - 归一化相当于给残差系统装了一个“稳压器”。

- 第三层：让梯度传播更顺
  - 尺度稳定以后，梯度也更容易稳定传播，不容易出现某层特别敏感、某层特别迟钝的问题。

把神经网络中的向量想成“信号”：
- 归一化前：信号有时太强，有时太弱
- 归一化后：信号强度统一了，后面模块更容易处理




## 公式
### LayerNorm公式
$$
y_i = \frac{x_i - \frac{1}{n}\sum_{i = 1}^{n}{x_i}}{\sqrt{(x_i - \frac{1}{n}\sum_{i = 1}^{n}{x_i})^2+\epsilon}} * \gamma + \beta
$$

子公式：
- 求均值：$\mu = \frac{1}{n}\sum_{i = 1}^{n}{x_i}$

- 求方差：$\sigma^2 = \frac{1}{n}\sum_{i = 1}^{n}{(x_i - \mu)^2}$

- 归一化：$\hat{x_i} = \frac{x_i - \mu}{\sqrt{\sigma^2 + \epsilon}}$

- 缩放+偏移：$y_i = \hat{x_i} * \gamma + \beta$ （LayerNorm公式简化显示）

### RMSNorm公式  
$$
y_i = \frac{x_i}{\sqrt{\frac{1}{n}\sum_{i = 1}^{n}{x_i^2} + \epsilon}} * \gamma + \beta
$$

子公式：
- 均方根RMS计算：$RMS(x) = \sqrt{\frac{1}{n}\sum_{i = 1}^{n}{x_i^2}}$

> 其中，$x_i$是输入向量的第$i$个元素，$n$是输入向量的维度，$\epsilon$是一个小常数，用于防止除以零，$\gamma$是一个可学习的缩放参数，$\beta$是可选偏移。
