# 环境搭建
参考项目地址：[minimind](https://github.com/jingyaogong/minimind.git)  
+ python版本：`Python 3.11.7`
+ 虚拟环境：`venv`
+ 其他配置参考：原项目`requirements.txt`  

## 下载初始化venv
### 安装
+ 安装venv：
    ``` shell
    $ powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```
+ 将venv路径添加到系统环境变量中，路径默认为`C:\Users\用户\.local\bin`
+ 检查是否安装成功：
  ``` shell
  $ uv --version
  ```
  出现版本号则安装成功

### 初始化
+ 创建虚拟环境：打开项目路径，终端输入
  ``` shell
  $ uv init [project_name]
  $ uv venv
  ```
+ 激活环境
  ``` shell
  $ .venv\Scripts\activate
  ```

+ 添加安装依赖
  打开`pyproject.toml`,将  
  ``` python
  dependencies = []
  ```
  中添加  
  ``` python
  dependencies = [
    "numpy>=2.3.4",
    "pandas>=2.3.3",
    "torch>=2.9.0",
    "transformers>=4.57.1",
    ]
  ```
  终端输入  
  ``` shell
  $ uv sync     # 安装依赖，配置虚拟环境
  ```  
+ 检查cuda和pytorch是否安装成功
  ``` python
  import torch
  print(torch.__version__)
  print(torch.version.cuda)
  print(torch.cuda.is_available())  #输出为True，则安装无误
  ```

# 搭建工程文件夹层次
+ `model`文件夹编写模型  
    + model.py
+ `trainer`文件夹用于训练
    + trainer_utils.py  编写通用型工具
    + trainer_pretrain.py  编写预训练工具
+ `dataset`文件夹用于处理数据集等相关资料
